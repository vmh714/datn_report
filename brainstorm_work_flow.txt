## BƯỚC 3: Đồng bộ ngược lại (Sync) lên Overleaf và Biên dịch

Sau khi AI Agent đã viết xong dưới máy và bạn đã đọc soát, verify lại nội dung (đảm bảo không bị "văn phong AI" và đúng kỹ thuật), bạn tiến hành đồng bộ lên Overleaf.

### Trường hợp 1: Nếu bạn dùng Git (Tự động & Chuyên nghiệp)

Mỗi lần viết xong một mục hoặc một chương, bạn chỉ cần mở Terminal dưới máy và chạy các lệnh Git cơ bản:

```bash
git add .
git commit -m "Update Chuong 1: Thêm nội dung mục 1.2 đã verify"
git push origin master

```

**Cơ chế biên dịch trên Overleaf:** Khi bạn `git push`, toàn bộ mã nguồn dưới máy sẽ được đẩy thẳng lên đám mây của Overleaf. Bạn chỉ cần mở trình duyệt Overleaf ra, dự án sẽ tự động cập nhật code mới nhất. Bạn chỉ việc bấm nút **Recompile** (hoặc bật chế độ Auto-compile) trên Overleaf để nó sinh ra file PDF hoàn chỉnh.

### Trường hợp 2: Nếu bạn dùng tài khoản Free (Không có Git)

Bạn có thể sử dụng một công cụ bên thứ ba tên là **`overleaf-sync`** (một tool viết bằng Python khá phổ biến trên GitHub).

1. Cài đặt qua terminal: `pip install overleaf-sync`
2. Sử dụng lệnh `ol s` để đồng bộ thủ công giữa thư mục local và dự án trên web Overleaf mà không cần qua cổng Git Premium.
