# FastAPI Route Generator

Standardizes API development.

## When to use this skill

- Use this when adding new REST API endpoints.
- This is helpful for maintaining consistent response formats.

## How to use it

1. Define the Pydantic models for request and response.
2. Implement the endpoint logic with appropriate HTTP methods.
3. Integrate with Database services (Postgres/Influx).
4. Add error handling and documentation (Swagger).
