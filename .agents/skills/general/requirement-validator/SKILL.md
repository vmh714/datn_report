# Requirement Validator

Ensures adherence to project constraints.

## When to use this skill

- Use this during code reviews or planning phases.
- This is helpful for preventing scope creep.

## How to use it

1. Compare the current task/code against the "Strict Scope" in `instruction.md`.
2. Verify the tech stack matches (e.g., Next.js App Router, Tailwind, Shadcn, FastAPI, ESP-IDF).
3. Identify any deviations or missing mandatory features.
4. Provide a report with "PASS/FAIL" status for each requirement.
