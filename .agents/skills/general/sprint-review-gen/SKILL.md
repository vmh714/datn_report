# Sprint Review Generator

Generates progress reports for stakeholders.

## When to use this skill

- Use this at the end of a milestone or sprint.
- This is helpful for tracking overall project health.

## How to use it

1. Analyze `task.md` for completed items (`[x]`).
2. Summarize the major features delivered.
3. List any open blockers or "in-progress" items.
4. Estimate remaining work based on the scope.
