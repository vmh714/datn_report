# Shadcn Component Builder

Builds UI components following the project's design system.

## When to use this skill

- Use this when building new UI elements for the Dashboard or Data Collection screens.
- This is helpful for maintaining visual consistency.

## How to use it

1. Identify the base Shadcn component needed.
2. Apply Tailwind classes for styling (Layout, Colors, Spacing).
3. Integrate Lucide-react icons where appropriate.
4. Ensure the component is accessible (ARIA labels, keyboard nav).
5. Export as a clean, reusable React component.
